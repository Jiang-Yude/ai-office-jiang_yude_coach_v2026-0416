## ⚡ 三分鐘快速啟動

### 步驟 1：開啟 Codex（或 Claude Code / Cowork）

開啟你已經安裝好的 AI 工具（**推薦零基礎用 Codex**）。

### 步驟 2：cd 到這個資料夾

在終端機（Codex 介面內）輸入：

```bash
cd "<這個資料夾的路徑>"
```

例如：

```bash
cd ~/Documents/AI辦公室基本款-v2026-05-17
```

### 步驟 3：跟 AI 說「咪卡在嗎」

AI 會讀取 `agents.md`，認識自己的工作環境，然後出場。

如果 AI 沒按腳本念招呼語、或自編一個情境化開場——**沒關係**，你直接看下方「你可以選三條路」自己決定要從哪開始。

## ⚠️ 注意事項

1. **這個資料夾本身就是你的 AI 工作環境**，不要刪、不要搬到 iCloud 外的奇怪地方
2. **macOS 放桌面或文件夾都可以**
3. **Windows 看到亂碼**請執行 `chcp 65001` 或改用 VS Code 開啟
4. **第一次跑可能會慢**——AI 在讀整個資料夾架構，正常

## 🆘 卡關了怎麼辦？

對 AI 說：

- 「**我卡住了**」→ AI 會問你卡在哪、幫你想辦法
- 「**幫我看看這個錯誤**」+ 貼出錯誤訊息 → AI 會幫你 debug
- 「**重新開始**」→ AI 會回到初始狀態

或者直接退出（`Ctrl+D` 或 `/quit`），重新進來再試。
