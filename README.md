# AI 辦公室 — 桌面版 AI Agent 打造你的 AI 辦公室

**解壓縮，打開，跟 AI 說「助理在嗎」，就開始了。**

由江江教練（姜育德）設計。把你的電腦變成 AI 辦公室，讓 AI 在你的檔案系統裡幫你做事。

---

## 這是什麼

一套可以直接使用的 AI 辦公室系統。下載後用 Claude Code、Cowork（反重力）或 OpenAI Codex 打開這個資料夾，AI 就會讀取裡面的設定，變成你的辦公室助理。

不用寫程式。不用學指令。乾淨的 Markdown 文件加上清楚的資料夾結構，就是整套系統。

---

## 核心概念

**你的文件就是你的系統。**

過去你寫一份操作手冊，只有人能讀。現在有了桌面版 Agent，同樣一份文件，AI 也能讀、也能照做。文件不再只是參考資料，而是可以被執行的系統。

AI 的各種模型就像厲害的顧問和員工，來來去去。重點不是去認識每一個模型，而是把你的工作系統講清楚——他們進來是學你的作業流程、來幫你打工的。

## 包含什麼

### 兩個角色

| 角色 | 功能 | 觸發方式 |
|------|------|---------|
| **AI 助教** | 教你怎麼用、解釋概念、引導操作 | 「這是什麼」「教我」「怎麼用」 |
| **雷總管** | 管看板、管檔案、管安全、記日誌 | 「開卡」「今天做什麼」「整理看板」 |

### 六個技能包

| 技能包 | 功能 |
|--------|------|
| **雷總管**（personal-director） | 看板管理、檔案管理、安全機制、工作日誌 |
| **AI 助教**（ai-teaching-assistant） | 11 個教學模組，從零開始引導 |
| **標籤管理**（tag-manager） | 幫文件加標籤、整理受控詞彙 |
| **圖片格式轉換**（image-converter） | HEIC 轉 JPG、PNG 轉 WebP 等 |
| **截圖改名**（image-renamer） | 把截圖的亂碼檔名改成中文 |
| **PDF 翻譯卡片化**（pdf-translation-cardifier） | 翻譯 PDF 並整理成重點卡片 |

### 內建觀念

- **3X4 資料管理法** — 3 種日記類型 x 4 種時效，決定東西記什麼、放哪裡
- **記憶管理六層模型** — 入口、技能、記錄、狀態、檢索、存放
- **知識留痕四層法** — 原始資料 → 關鍵標記 → 轉折說明 → 整理版輸出
- **安全機制** — 刪除進垃圾桶、操作前確認、禁止直接刪除
- **工作看板** — 任務追蹤中心，AI 每天第一個讀的地方

## 怎麼開始

### 第一步：下載

點右上角 `Code` → `Download ZIP`，解壓縮到你想放的位置。

### 第二步：選一個工具打開

| 工具 | 特色 | 適合誰 |
|------|------|--------|
| **Claude Code** | 最完整，終端機操作 | 願意學指令的人 |
| **Cowork（反重力）** | 圖形化介面，安全機制好 | 文書處理為主的人 |
| **OpenAI Codex** | 免費版可用 | 先熟悉手感的人 |

用工具打開這個資料夾（不是單一檔案，是整個資料夾）。

### 第三步：跟 AI 打招呼

跟 AI 說：

> 助理在嗎

AI 會自動讀取設定，引導你完成初始化：建立工作台、看板、垃圾桶等基礎結構。

### 第四步：開始用

- 「開卡：買牛奶」→ 新增一筆待辦
- 「今天做什麼」→ 看待辦清單
- 「教我」→ AI 助教開始教學
- 「設定」→ 看功能開關

## 搭配 Obsidian 使用（推薦）

AI 負責改檔，[Obsidian](https://obsidian.md/) 負責看檔。改完用 Obsidian 打開看結果。

安裝 Obsidian 後，用它打開同一個資料夾，再裝 Kanban 外掛，看板就會變成視覺化的拖拉介面。

## 常見問題

**Q：我完全不懂 AI，可以用嗎？**
A：可以。AI 助教會從零開始教你，一步一步來。

**Q：會不會把我的檔案搞壞？**
A：AI 辦公室內建安全機制。刪除的東西都進垃圾桶，重要操作前會先確認。建議先在這個資料夾裡練習，不要一開始就放重要文件。

**Q：我可以自己加功能嗎？**
A：可以。跟 AI 說「幫我做成技能包」，AI 會幫你把工作方法封裝成新的技能包。

**Q：支援哪些 AI 工具？**
A：Claude Code、Cowork（反重力）、OpenAI Codex。未來有新工具支援 Markdown 技能包格式的，都可以用。
