# 👋 從這裡開始

歡迎進入 **AI 辦公室**。
這是一份「給 AI 用的工作環境」——所有檔案都放在這個資料夾裡，AI 來「上工」就找這裡。

---

## ⚡ 三分鐘快速啟動

### 方法 A：直接從 GitHub 拷貝（推薦）

最簡單——讓 AI 幫你做：

1. 在文件夾建一個新資料夾，例如「**我的 AI 辦公室**」
2. 開啟 Codex，cd 進該資料夾
3. 跟 AI 說：「**幫我從這個 GitHub 把 AI 辦公室拷貝下來**」
   `https://github.com/Jiang-Yude/ai-office-jiang_yude_coach_v2026-0416`
4. AI 會自動 `git clone` 整套放進來

**好處**：跳過下載 zip 跟解壓縮、永遠拿到最新版、之後也能用 `git pull` 更新。

### 方法 B：下載 ZIP（不會用 git 的人）

1. 點 GitHub 頁面綠色按鈕「**Code → Download ZIP**」
2. 解壓縮到桌面或文件夾
3. 開啟 Codex，cd 進該資料夾

### 方法 C：雲端硬碟下載（學員專屬）

如果你是江江教練的學員，老師會在 LINE 群給雲端硬碟連結，下載資料夾即可。

## 第一句話跟 AI 說

拷貝下來後，跟 AI 說：

> **「咪卡在嗎」**

AI 會讀取 `agents.md`，認識自己的工作環境，然後出場。

如果 AI 沒按腳本念招呼語、或自編一個情境化開場——**沒關係**，你直接看下方「你可以選三條路」自己決定要從哪開始。

## 🐱 你可以選三條路

> 嗨，我是江江教練設計的 AI 助理。
> 歡迎進入 AI 辦公室。

跟 AI 對話時，三條路任選一條：

### 1. 🎯 現在就有想處理的文件或專案嗎？

→ 「**我有一份 [XXX]，幫我處理**」
→ AI 直接動手，一邊做一邊告訴你發生了什麼事。

### 2. 🗺️ 想先認識這裡有哪些助理可以幫你？

→ 「**這裡有哪些助理可以幫我？**」
→ AI 介紹一輪，讓你以後知道該叫誰。

### 3. 💭 想慢慢來也可以

→ 「**我想聊聊我手上的工作**」
→ AI 陪你聊，再設計適合你的方式。

## 📚 想升級到最新版？

說「**檢查更新**」或「**有沒有新版**」，AI 會自動檢查更新通道並引導你 OTA 更新。

## ⚠️ 注意事項

1. **這個資料夾本身就是你的 AI 工作環境**，不要刪、不要搬到 iCloud 外的奇怪地方
2. **macOS 放桌面或文件夾都可以**
3. **Windows 看到亂碼**請執行 `chcp 65001` 或改用 VS Code 開啟
4. **第一次跑可能會慢**——AI 在讀整個資料夾架構，正常

---

## 🆘 卡關了怎麼辦？

對 AI 說：

- 「**我卡住了**」→ AI 會問你卡在哪、幫你想辦法
- 「**幫我看看這個錯誤**」+ 貼出錯誤訊息 → AI 會幫你 debug
- 「**重新開始**」→ AI 會回到初始狀態

或者直接退出（`Ctrl+D` 或 `/quit`），重新進來再試。
